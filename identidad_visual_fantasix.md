# Identidad Visual y UX: Fantasix

## 1. Concepto de Marca
- **Nombre:** Fantasix (Fantasy + R6).
- **Estilo:** Táctico, Militar-Tecnológico, Minimalista. Inspirado en el HUD de Rainbow Six Siege.

## 2. Paleta de Colores "Premium Tactical" (Tailwind Tokens)
- **Background (Deep Void):** `#0A0B10` (Un azul/morado casi negro, muy profundo y elegante para el fondo).
- **Surface (Card):** `#151226` (Un morado noche para las tarjetas de los jugadores y contenedores, resalta sobre el fondo).
- **Primary (Champion Gold):** `#FBBF24` (Un dorado vibrante para botones principales, iconos de rango, coronas o destacar al MVP. Transmite prestigio).
- **Secondary (Neon Purple):** `#8B5CF6` (Un morado eléctrico para barras de progreso, acentos visuales y enlaces).
- **Text Primary:** `#F8FAFC` (Blanco puro/azulado para máxima legibilidad).
- **Text Secondary:** `#94A3B8` (Gris azulado para estadísticas y textos de apoyo).

## 3. Tipografía
- **Títulos (Display):** 'Rajdhani', sans-serif (Peso 600-700). Estilo industrial y cuadrado.
- **Cuerpo y Datos:** 'Inter', sans-serif.
- **Estadísticas:** 'Roboto Mono' (Para que los números en tablas estén perfectamente alineados).

## 4. Elementos de Diseño
- **Bordes:** Esquinas ligeramente redondeadas (`rounded-lg`) o cortes angulares de 45 grados en botones.
- **Efectos:** Glassmorphism sutil en tarjetas (fondo con opacidad y desenfoque).
- **Iconografía:** Luces de estado (punto verde para "online", rojo para "eliminado").
